# 🛍️ Perfume Shop App

A simple and responsive **Perfume Store web application** built using **HTML, CSS, JavaScript, and Bootstrap**.
The project focuses on displaying products in a clean UI with smooth user experience and dynamic interaction.

---

## 🚀 Live Demo 🔗

[https://3tourshop.netlify.app/](https://3tourshop.netlify.app/)

---

## 📌 Features

* 🧴 Display perfumes in a structured product layout
* 🔍 Clean and modern UI for browsing products
* 📱 Fully responsive design (mobile / tablet / desktop)
* ⚡ Fast and lightweight front-end performance
* 🎨 Styled using Bootstrap with custom enhancements
* 🧠 Basic dynamic behavior using JavaScript

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Custom styling and layout control
* **Bootstrap** – Responsive grid and UI components
* **JavaScript (ES6)** – Interactive behavior
* **DOM Manipulation** – Dynamic rendering of elements

---

## 📦 Project Type

Frontend E-commerce UI (Static Store Front)

---

## 📎 Notes

* No backend or database integration
* Product data handled on the frontend
* Designed for UI/UX practice and portfolio presentation
